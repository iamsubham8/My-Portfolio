This game is built using JavaScript and allows the player to control a character using the arrow keys or the spacebar. The objective is to avoid oncoming obstacles by jumping over them. Each successful avoidance increases the player's score, which is displayed on the screen. The game features background music and sound effects to enhance the experience. If the player collides with an obstacle, the game ends, and the user must reload the page to restart.

üì±Functioning of Application

User Interface

The game displays a character, obstacles, and a real-time score on the screen. The background features dynamic elements and music to keep the player engaged.

User Input

Players use the arrow keys or spacebar to make the character jump and avoid obstacles. Input detection listens for keydown events to trigger the jump animation.

Obstacle Movement

Obstacles continuously move toward the character, simulating a scrolling environment. Their speed can increase over time to raise difficulty.

Collision Detection

The application continuously checks for collisions between the character and obstacles. If a collision is detected, the game ends immediately.

Score Tracking

For every successful avoidance (i.e., when an obstacle passes without a collision), the score increments. The current score is displayed on the screen.

Audio Integration

Background music plays throughout the game, and sound effects trigger during key events (e.g., jump, score increment, or collision)..

Game Over Logic

When a collision occurs, the game stops. The user must reload the page to restart, providing a simple reset mechanism.
