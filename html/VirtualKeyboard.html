AI Virtual Keyboard ⌨️

The AI Virtual Keyboard provide type letters from detecting our hands and gestures, with the help of utilizes computer vision techniques. It is a software based program that allows users to input characters using gestures in device without using physical keyboard.

✨Key Features

  • Virtual keyboard interface display on the screen.
  • Hand tracking-based key pressing for typing.
  • Real-time interaction with a virtual keyboard.
  • Customizable for different keys.

🛠️Technologies Used

  • OpenCV (Open Source Computer Vision)
  • Hand Tracking Module (Custom or Mediapipe)
  • Numpy
  • Pynput
  • Other libraries (math, time, etc.)

⚙️Functioning of Application

User Interface

The on-screen keyboard that shows feedback for gesture input.

Input Gesture

The input layer where users interact with the virtual keyboard, interpreting gestures as input.

Hand Tracking

Once the user places their hand within the interaction space, the system captures real-time data about the position and movement of the hands and fingers.

Input Processing

Interpreting coordinates involves analyzing and understanding the user's coordinates to accurately translate them into corresponding text.

Output Generation

The recognized letters based on hand tracking and coordination are displayed on the screen.

🏆Outcomes

  • Contactless Solutions (Healthcare & Laboratories)
  • Interactive Gaming and Entertainment
  • Advancements in Smart TV Control
  • Assistive Technology for People with Disabilities

📝Conclusion

The AI Virtual Keyboard project successfully demonstrates the potential of touchless, gesture-based interaction with users. It offers an innovative and efficient way to enhance user interaction with digital devices. This approach not only improves speed and accuracy in text input but also reduces the physical effort needed for traditional typing.
